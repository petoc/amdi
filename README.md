# Apple mobile device identifiers

Identifiers and product names for Apple mobile devices.

## Usage

```go
import "github.com/petoc/amdi"
```

```go
amdi.Is("iPhone1,1")     // true
amdi.Name("iPhone1,1")   // iPhone
```
